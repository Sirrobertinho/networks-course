### Задача 2 (3 балла)
Рассмотрим раздачу файла размером $F = 15$ Гбит $N$ пирам. Сервер имеет скорость отдачи $u_s = 30$
Мбит/с, а каждый узел имеет скорость загрузки $d_i = 2$ Мбит/с и скорость отдачи $u$. Для $N = 10$, $100$
и $1000$ и для $u = 300$ Кбит/с, $700$ Кбит/с и $2$ Мбит/с подготовьте график минимального времени
раздачи для всех сочетаний $N$ и $u$ для вариантов клиент-серверной и одноранговой раздачи.

#### Решение
$\text{Для клиент-серверной раздачи минимальное время равно: } T_{min} = max(\dfrac{N F}{u_s}, \dfrac{F}{d_{min}}).$  
$\text{Для одноранговой раздачи минимальное время равно: } T_{min} = max(\dfrac{F}{u_s}, \dfrac{F}{d_{min}}, \dfrac{N F}{u_s + N u}).$  
$\text{Для клиент-серверной раздачи формула доказывается в задаче 3. Для одноранговой: } \dfrac{F}{u_s} - \ \text{время необходимое серверу для загрузки целого файла хотя бы раз}, \ \dfrac{F}{d_{min}} - \ \text{время загрузки файла на один самый медленный пир}, \ \dfrac{N F}{u_s + N u} - \ \text{суммарная информация / максимальная скорость распространения данных во всей системе.}$  
$\text{То что минимальное время хотя бы максимум это очевидно. Схемы необходимые для этого примерно такие: сервер равномерно загружает файл на пиры, которые распространяют друг с другом части файла.}$

### Задача 3 (3 балла)
Рассмотрим клиент-серверную раздачу файла размером $F$ бит $N$ пирам, при которой сервер
способен отдавать одновременно данные множеству пиров – каждому с различной скоростью,
но общая скорость отдачи при этом не превышает значения $u_s$. Схема раздачи непрерывная.
1. Предположим, что $\dfrac{u_s}{N} \le d_{min}$.
   При какой схеме общее время раздачи будет составлять $\dfrac{N F}{u_s}$?
2. Предположим, что $\dfrac{u_s}{N} \ge d_{min}$. 
   При какой схеме общее время раздачи будет составлять  $\dfrac{F}{d_{min}}$?
3. Докажите, что минимальное время раздачи описывается формулой $\max\left(\dfrac{N F}{u_s}, \dfrac{F}{d_{min}}\right)$?

#### Решение
$\text{3.1 Введём обозначение: } v_i(t) - \ \text{скорость отдачи } i-\text{ому пиру в момент времени } t, \ t_i - \ \text{время, которое понадобилось } i-\text{ому пиру для получения всех } F \ \text{бит. Считаем, что скорость после получения всего равна 0.}$  
$\text{Теперь давайте докажем, что хотя бы один пир получит все биты не раньше, чем через время } \dfrac{N F}{u_s}.$  
$\text{Общее число битов, полученных } i-\text{ым пиром равно интегралу скорости по времени, т.е. }  \int_{0}^{t_i} v_i(t) \ dt.$  
$\text{Пусть } T = max_{i}(t_i). \ \text{Тогда, т.к. мы считаем скорость нулевой после получения всех бит, то можно считать, что } i-\text{ый пир получил бит: } \int_{0}^{T} v_i(t) \ dt. $  
$\text{Сложим все интегралы: } \int_{0}^{T} \sum_{i=1}^{N} v_i(t) \ dt \stackrel{\text{по условию}}\leq \int_{0}^{T} u_s \ dt = u_s \cdot T.$  
$\text{С другой стороны, если сложить все полученные биты, то получится } F \cdot N.$  
$\Rightarrow F \cdot N \leq u_s \cdot T \Leftrightarrow T \geq \dfrac{N F}{u_s}. \ \text{Значит раз } T = max_{i}(t_i), \ \text{то есть время у какого-то пира, которое равно } T, \ \text{и оно } \geq \dfrac{N F}{u_s}.$  
$\text{Схема, при которой время будет именно таким будет, когда скорость отдачи всем пирам: } \dfrac{u_s}{N}. \ \text{Это возможно, т.к. минимальная скорость получения всех пиров } d_{min}, \ \text{а по условию: } \dfrac{u_s}{N} \leq d_{min}.$  
$\text{Время, соответственно, будет у каждого пира: } \dfrac{F}{\frac{u_s}{N}} = \dfrac{N F}{u_s}.$  
$\text{3.2 Заметим, что есть пир со скоростью получения } d_{min}. \ \text{Для него время, за которое он сможет получить все биты, не превосходит: } \dfrac{F}{d_{min}}.$  
$\text{Схема, при которой время будет именно таким будет, если всем пирам скорость передачи будет } \geq d_{min} \ \text{(можно сделать прям равно } d_{min} \ \text{у каждого пира)}. \ \text{Это возможно, т.к. минимальная скорость получения: } d_{min}, \ \text{а суммарная скорость отдачи: } N \cdot d_{min} \leq u_s, \ \text{т.к. по условию: } d_{min} \leq \dfrac{u_s}{N}.$  
$\text{3.3 Рассмотрим два случая:}$  
$\text{1-ый случай: } \dfrac{N F}{u_s} \geq  \dfrac{F}{d_{min}} \Leftrightarrow \dfrac{u_s}{N} \leq d_{min}. \ \text{Тогда по пункту 3.1 есть пир со временем хотя, бы } \dfrac{N F}{u_s}, \ \text{и при этом общее такое время достигается, значит минимальное время } \dfrac{N F}{u_s} = \max\left(\dfrac{N F}{u_s}, \dfrac{F}{d_{min}}\right).$  
$\text{2-ой случай: } \dfrac{N F}{u_s} \leq  \dfrac{F}{d_{min}} \Leftrightarrow \dfrac{u_s}{N} \geq d_{min}. \ \text{Тогда по пункту 3.2 есть пир со временем хотя, бы } \dfrac{F}{d_{min}} \ \text{и при этом общее такое время достигается, значит минимальное время } \dfrac{F}{d_{min}} = \max\left(\dfrac{N F}{u_s}, \dfrac{F}{d_{min}}\right).$  
$\text{Мы рассмотрели все случаи и формула работает. } \blacksquare$
